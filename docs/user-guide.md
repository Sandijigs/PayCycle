# PayCycle User Guide

## How to Test PayCycle in 5 Minutes

### For Merchants (Create a Subscription Plan)

1. Install Freighter wallet extension
2. Switch to Testnet in Freighter settings
3. Fund your wallet via Friendbot (link in app)
4. Go to paycycle.vercel.app
5. Connect wallet → "Create Plan" → set name, amount, interval
6. Share your plan link with potential subscribers

### For Subscribers

1. Install Freighter wallet extension
2. Switch to Testnet
3. Fund wallet via Friendbot
4. Visit a plan link (or browse plans in the app)
5. Click "Subscribe" → approve spending cap in Freighter
6. Done! Payments auto-execute each cycle.
7. Pause or cancel anytime from "My Subscriptions"
